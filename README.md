# lifelong-learning
みんなで叶える夢プロジェクト

それらに必要だと思う事や、それらの記録や過程結果に至る迄のリポジトリ。

・何故作る事にしたか
　学生時代から計画していた夢の土台がやっと整った。
　自分自身、勉強していて、まとめた思考ノートやコードすらも紛失するし、ならいっその事残そうと思い至った。
　何年も何年も悩み続けて、形に残そうと思えて決意が固まった為。
 　見てくれてるか、分からないけど、過去に自分を信じて、護ってくれた人達みんなに、自分の進捗を目に見える化をする為。
書き連ねたらキリが無い

・教わった恩師の言葉を忘れずに努力し続けたいし、一個ずつ実現の為に、生涯学習。

・あの日の約束を守る。
　行動しなかったり、夢を実現出来なかったら、自分を信じたみんなを裏切る事になるし、それに、過去を否定すれば、今の自分は、今ここに自分は、存在しない。誓約の意味でもある。

 ・最後に、学びたいと思う人へ
 　もしこのリポジトリが目に入ったら、少しでも貴方の役に立ちますように、ただより良くなって欲しいその願いを込めて。
